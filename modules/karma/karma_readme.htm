Karma Module

Created by: DB and Yankidank
Last Updated: 08/11/2011 Created on: 08/20/2010
Requires: Nothing

Description

The Karma module recalculates user karma scores when common actions are performed, such as when a user submits an article, leaves a comment, or performs a voting action. Karma scores are a good indicator of how active an account is on a Pligg website, and Pligg CMS makes use of karma scores when displaying the "Top Users" page.
