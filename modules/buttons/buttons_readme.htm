External Vote Button Module

Created by: DB and Yankidank
Last Updated: 03/31/2011
Created on: 10/12/2010

Description

This module will create Vote buttons on other sites.

Settings

The module's settings page is where you customize the display options for the external vote button. This module supports 2 different buttons, generally used as a large and small versions of the button.

From the module settings area you can define the CSS styling,

The CSS used to style the EVB buttons is stored in the cache file /cache/evb.css. You can edit this file from the module's settings page.
