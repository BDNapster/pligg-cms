Twitter Module

Created by: DB and Yankidank
Last Updated: 03/28/2013
Created on: 12/16/2010

Description

This module allows members to register and login using Twitter account.

Installation

For Pligg:

  1. Upload the /modules directory that this download contains to where your Pligg site is installed. This will add the "twitter" directory to your /modules folder.
  2. Log in to a "admin" level account on your Pligg website.
  3. Navigate to the /admin directory.
  4. Go to the "Modules" tab from the admin panel.
  5. Click on the "Uninstalled Modules" sub-tab to view all of your uninstalled modules.
  6. Click the install button for the row with the "Twitter Module" title.
  7. Navigate to the "Installed Modules" tab, then click on the "Settings" button for the "Twitter Module".
  8. Follow the on-screen directions for setting up a Twitter API key.
  9. If you wish to use Bit.ly shortened URLs, fill in the 2 bit.ly fields. Find your Bit.ly API key by going to http://bit.ly/a/account.
  10. If your site is using a template designed for 1.1.3 or below you will need to make the following adjustments:
    • Open your template's header.tpl file
    • Find: {checkActionsTpl location="tpl_pligg_login_link"}
    • Move the module hook from line b outside of the statement: {if $user_authenticated neq true} . Usually this means moving it down a couple lines just after {/if}

For Twitter:

  1. After logging into a Twitter account that you want to represent your site, navigate to http://dev.twitter.com/apps/new
  2. Fill in the Application name (will be seen when users connect their account), description, website (Pligg root), and Organization.
  3. For callback URL, enter the URL found on the Twitter module settings page. It should be something like:
       http://www.DOMAIN.TLD/DIR/modules/twitter/tweet.php?mode=callback
  4. After saving your initial settings, visit the "Settings" tab.
  5. Change the "Application Type" access to "Read & Write", and save the settings.
  6. Go back to the "Details" tab, and find the Twitter Application Consumer Key and paste it into the first field in the Twitter module settings page.
  7. On the same page, Find the Twitter Application Consumer Secret key and paste that into the second field on the settings page.
